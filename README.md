# Armor Ball
A small roller ball game example made with [Blender](https://www.blender.org/) 4.5 and [Armory3D](https://armory3d.org/engine/) 2026.02.

Made with scalability in mind and featuring:
- Gameplay with basic physics interactions
- Coin collectables
- Scene transitions
- 3 game levels
- Music
- Sound effects
- Assets library usage
- UI with dynamic labels and buttons

## Pre-requisites & Setup
[Armor 64's pre-requisites](https://github.com/moisesjpelaez/armor_64#pre-requisites) must be met for N64 builds.

Use `git clone --recursive https://github.com/moisesjpelaez/arm_ball.git` to clone the project with all the libraries.
